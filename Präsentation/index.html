Title und Description

  • URL der Website
  • Keywords im Title die den Nutzer vermitteln worum es geht (Überschrift)
  • kurze "Beschreibung" um den Inhalt der Seite zu erläutern

Klickrate

Die Klickrate bezeichnet den Prozentsatz der User, die nach einer Google-Suche auf ein Suchergebnis klicken.

Anzahl der Klicks / Anzahl der Suchanfragen x 100 = Klickrate

Sitelinks

Rich Snippets

Sind Auszeichnungen von Google mit besonderen Informationen. Bringen Vorteile wie: Auffälligkeit, Vertrauen in die Website, besseres Ranking.

noindex-Anweisung

  • urheberrechtlich geschützte Inhalte
  • z.B. das Impressum um Namen nicht im Index auftauchen zu lassen

nofollow-Anweisung

meta name="robots" content="index,follow" Steht dafür, dass der Googlebot Links folgen darf. Gegenteil dafür wäre nofollow was genau das nicht erlaubt.

nosnippet-Anweisung

meta name"robots" content="nosnippet" Verhindert, dass Google die Description anzeigt.

noarchive-Anweisung

meta name"robots" content="noarchive" Ist sinnvoll, wenn du nicht möchtest, dass Google eine Kopie deiner URL cached. Ressourcensparend dadurch das keine älteren Versionen gespeichert werden. Hat keine negativen Auswirkungen auf das Ranking.
